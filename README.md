## 1. User Flow

```mermaid
flowchart TD
    A([🟢 Запуск]) --> B[Экран: Главное меню\n
    1. Новая игра 
    2. Статистика 
    3. Выход]
    B --> C{Выбор}
    C -->|1| D[Экран: Выбор сложности\neasy/medium/hard]
    C -->|2| E[Экран: Статистика с средним временем]
    C -->|3| Z([🔴 Завершение])
    D --> F[Экран: Игровое поле 9*9 
    + таймер 
    + подсказка]
    E --> B
    F --> G{Ввод пользователя}
    G -->|"ряд столбец число"| H{Валидация}
    G -->|hint| I[Подсказка: 
    координаты + значение]
    G -->|advice| J[Совет по стратегии]
    G -->|exit| K[Подтверждение сдачи]
    G -->|rules| L[Экран: 
    Правила + управление]
    I --> F
    J --> F
    K -->|y| M[Экран: 
    Решение с подсветкой всех клеток]
    K -->|n| F
    L --> F
    H -->|✅ Верно| N[Обновление поля
     + зелёная подсветка]
    H -->|❌ Ошибка| O[Красный текст 
    + подсветка клетки\nBack.WHITE+Fore.RED]
    N --> P{Поле заполнено?}
    O --> F
    P -->|Нет| F
    P -->|Да + совпадает с solution| Q[Экран: ПОБЕДА 
    + ASCII-арт 
    + время]
    P -->|Нет| R[Сообщение: 
    Проверьте красные клетки]
    Q --> S[Сохранение в sudoku_stats.json]
    R --> F
    M --> S
    S --> B
